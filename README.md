# RenForge (v1.0.0)

**RenForge** — это специализированный графический инструмент (GUI) для автоматизированного извлечения ресурсов и декомпиляции скриптов игр на движке **Ren'Py**. Программа объединяет возможности популярных консольных утилит в едином, интерфейсе.

## Ключевые особенности

- **Standalone-решение:** Программа полностью автономна. Для работы **не требуется** установка Python или внешних библиотек.
- **Распаковка RPA:** Поддержка всех версий архивов Ren'Py (RPA-1.0, 2.0, 3.0, 3.2, 4.0).
- **Декомпиляция RPYC:** Преобразование скомпилированных файлов `.rpyc` в исходный код `.rpy` с помощью актуальных алгоритмов.
- **Управление переводами:** Встроенная система сканирования проекта и база данных для работы с локализацией.

---

## Архитектура

1.  **Frontend:** Vue 3 (Composition API) + Vite — пользовательский интерфейс.
2.  **Backend:** Rust (Tauri 2.0) — управление процессами, работа с файловой системой и SQLite базой данных.
3.  **Sidecars (Побочные процессы):** Утилиты `unrpa` и `unrpyc` были модифицированы и скомпилированы в независимые исполняемые файлы (`.exe`) с помощью PyInstaller. 

---

## Установка

Программа распространяется в двух форматах для Windows (x64):

### 1. Установщик (Recommended)
Скачайте `RenForge_v1.0.0_Windows_Installer.msi` со страницы релиза. Программа будет установлена в систему с созданием ярлыков и регистрацией в реестре для корректного удаления.

### 2. Портативная версия
Скачайте `RenForge_v1.0.0_Portable.zip`. Распакуйте содержимое архива в любую папку и запустите файл `RenForge.exe`. 

---

## Инструкция по использованию

1.  **Выбор проекта:** Нажмите «Выбрать папку» и укажите путь к корневой директории игры или папке `game`.
2.  **Сканирование:** Программа автоматически определит наличие RPA-архивов и скомпилированных скриптов.
3.  **Распаковка:** Используйте функцию «Распаковать ресурсы» для извлечения изображений, музыки и скриптов из архивов.
4.  **Декомпиляция:** После распаковки нажмите «Вскрыть скрипты», чтобы получить редактируемые `.rpy` файлы.
5.  **Локализация:** Используйте вкладку сканера для поиска непереведенных строк и работы с базой данных переводов.
