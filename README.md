# RenForge (v1.1.0)

**RenForge** — это специализированный графический инструмент (GUI) для автоматизированного извлечения ресурсов, декомпиляции скриптов и комплексной локализации игр на движке **Ren'Py**. Программа объединяет возможности консольных утилит и полноценную среду для перевода в едином интерфейсе.

## Ключевые особенности

- **Standalone-решение:** Программа полностью автономна. Для работы **не требуется** установка Python или внешних библиотек.
- **Распаковка и декомпиляция:** Поддержка всех версий архивов RPA и актуальных алгоритмов декомпиляции `.rpyc` в исходный код `.rpy`.
- **Визуальный редактор перевода:** Построчное редактирование диалогов и строк интерфейса с проверкой целостности тегов и переменных.
- **Интеграция с ИИ (LLM):** Пакетный перевод строк через локальные нейросети (Ollama) или буфер обмена.
- **Медиа-менеджер:** Инструменты для локализации изображений и аудиофайлов (озвучки) с привязкой к игровым скриптам.
- **Карта проекта:** Визуализация структуры игры в виде интерактивного графа файлов.
- **Runtime-патчинг:** Функция внедрения патча в игру для форсированного выбора языка и поддержки горячей перезагрузки (Shift+R).

---

## Архитектура

1.  **Frontend:** Vue 3 (Composition API) + Vite — модульный интерфейс на компонентах.
2.  **Backend:** Rust (Tauri 2.0) — высокопроизводительное управление файловой системой и процессами.
3.  **Database:** SQLite — хранение прогресса перевода, статистики файлов и пользовательского глоссария.
4.  **Sidecars (Побочные процессы):** Утилиты `unrpa` и `unrpyc`, скомпилированные в независимые исполняемые файлы.

## Установка

Программа распространяется в двух форматах для Windows (x64):

### 1. Установщик (Recommended)
Скачайте `RenForge_v1.1.0_Windows_Installer.exe` со страницы релиза. Программа будет установлена в систему с возможностью корректного удаления.

### 2. Портативная версия
Скачайте `RenForge_v1.1.0_Portable.exe`. Запустите файл из любой директории для начала работы.

## Инструкция по использованию

1.  **Выбор проекта:** Нажмите «Выбрать папку» и укажите путь к корневой директории игры (там, где находится `.exe`).
2.  **Подготовка:** Используйте графическую карту для распаковки RPA-архивов и вскрытия скомпилированных скриптов.
3.  **Генерация:** Создайте структуру перевода для выбранного языка через встроенный движок Ren'Py.
4.  **Перевод:** Используйте визуальный редактор для работы с текстом. Поддерживается глоссарий и пакетная обработка через ассистента.
5.  **Медиа:** В разделах «Изображения» и «Аудио» выполняйте подмену ресурсов для локализации графики и звука.
6.  **Тестирование:** Внедрите патч и запустите игру. Используйте Shift+R в окне игры для мгновенного применения изменений без перезапуска.
